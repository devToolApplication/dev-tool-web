# component-file-standard

Component phải tách `.ts`, `.html`, `.css` khi có template/style đáng kể. Không inline template/style cho feature/shared component mới trừ trường hợp cực nhỏ đã có convention.

## Required Checks

- Đọc tài liệu này trước khi sửa khu vực liên quan.
- Cập nhật docs nếu folder, route, API contract hoặc shared boundary thay đổi.
- Không move/xóa path public khi chưa có compatibility wrapper hoặc alias.
